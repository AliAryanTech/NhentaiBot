# NhentaiBot

## Requirements

```
Python >=3.9.0
Git
Virtualenv
```

## Setup

- Clone the Repository

  ```
  git clone git@github.com:knight-byte/NhentaiBot.git
  cd NhentaiBot
  ```

- Setup a Virtualenv
  ```
  virtualenv .
  # start virtual env
  source bin/activate
  ```
- Installing requirements
  ```
  pip install -r requirements.txt
  ```
- Setting Environment variable

  ```
  export bot_token=<your bot token>

  ```

- Start Bot
  ```
  python -m nhentaiBot
  ```
